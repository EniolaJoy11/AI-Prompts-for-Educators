# AI Prompts for Educators

A collection of practical AI prompts designed to help educators save time, improve lesson planning, create resources, and streamline classroom tasks.

This contains a growing collection of **AI prompts designed to support educators** in lesson planning, assessment creation, student feedback, and classroom management.
These prompts are created to **save time**, **improve productivity**, and **enhance teaching efficiency** using AI tools.


Educators spend many hours planning lessons, writing reports, and creating resources.
This project provides **ready-to-use prompts** that make these tasks faster and easier.
